## 卷首语

为什么要学 Git ？问得好。也许你听说过它强大的版本管理功能，想要亲身尝试；或者跟你一起写代码的同学发现你不会 Git 然后把这个链接丢给了你。
不管如何，既然你已经打定主意学习 Git !!不管是主动还是被动!!，你一定需要理解它是怎样运作的。

本系列文章可以帮助你快速理解什么是 Git 、为什么用 Git 以及如何用 Git 。
