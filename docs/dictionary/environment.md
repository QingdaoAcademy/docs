## 环境

在编程中，“环境”通常指的是==软件开发和运行所需的各种条件和配置的集合==。这些条件和配置可以包括操作系统、编程语言的版本、依赖库、工具链、编辑器设置等。只有这些方面都满足要求，代码才能正确地运行。

## 包管理器

包管理器是在电脑中自动安装、配制、卸载和升级软件包的工具组合。

你可以简单理解为，包管理器可以帮你轻松地完成一些特定软件的安装，免去你搜索官网、下载、解压缩、安装等等繁琐步骤。

包管理器可以分成两类：

1. 系统级包管理器：用于管理操作系统层面的软件包，例如 MacOS 的 Homebrew、Debian 系 Linux 的 apt 等。这些包管理器通常用于安装软件，你可以使用他们来安装 Python、Git 等。

2. 语言级包管理器：用于管理特定编程语言的库和模块，例如 Python 的 pip、JavaScript 的 npm 等。这些包管理器通常用于安装和管理编程语言的第三方库。
